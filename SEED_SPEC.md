# SEED Specification

SEED is a thinking protocol.

Core Loop

Observe
Externalize
Reframe
Decide
Act
Repeat

Purpose

Return agency to the human.

Principles

• clarity over comfort  
• multiple perspectives  
• structure over ideology  
• human agency first

## Structure

SEED grows as a thinking ecosystem.

Seed → Forest → World Tree

Seed  
The smallest unit of thought.

Forest  
A network of connected seeds.

World Tree  
A network of forests forming a shared structure.
